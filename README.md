# 🔵 Cyber Agar: 球球大作战 (Cyber Edition) 🌈（不够完善）

嘿！欢迎来到我的球球世界！这是一个基于 **C++** 和 **EasyX** 图形库开发的 2D 竞技生存游戏。

在这里，你是一颗小小的蓝色星球，目标只有两个：**吃掉别人，或者被别人吃掉！** 😈

---

## 🌟 游戏亮点 (Features)

* **超大地图**：8000x8000 的巨型赛场，让你跑个痛快！
* **平滑摄像机**：自带“阻尼感”的丝滑镜头跟随，告别生硬切换。
* **炫酷特效**：
    * ✨ **粒子喷射**：冲刺时有尾气，吃掉敌人时有大爆炸！
    * 🌈 **赛博配色**：每一颗球球和食物都有亮眼的随机配色。
* **聪明(狡猾)的 AI**：大球会追你，小球会逃跑，AI 甚至还会躲避讨厌的绿色刺球！
* **多样道具**：
    * 💎 **黄金球**：一口气长胖 10kg！
    * ⚡ **加速球**：开启疯狂疾走模式。

---

## 🎮 怎么玩呢？ (How to Play)

1.  **移动**：使用键盘方向键 `↑` `↓` `←` `→` 控制你的球球。
2.  **冲刺**：按住 `Shift` 键急速前进！(温馨提示：冲刺会消耗体重哦，别把自己跑瘦了~)
3.  **通关目标**：吃掉比你小的东西，避开绿色的刺球，当体重达到 **500kg** 时，你就是全场最强王者！🏆

---

## 🛠️ 如何在你的电脑上运行？ (Setup)

**下载本体**：直接下载本仓库中的 `Ball.exe` 即可运行。


---

## 💌 小贴士 (Tips)

* 当你变大的时候，移动速度会变慢，这时候利用 `Shift` 冲刺伏击小球是致胜关键！
* 绿色的刺球（Virus）是胖子们的噩梦，一旦碰到，你会被炸得“四分五裂”！
